Support & FAQ

How does Nudge detect doom-scrolling?

Nudge uses Apple's Screen Time API to analyze usage patterns in real-time, identifying when scrolling becomes repetitive or extended beyond normal browsing.

Is my usage data private?

Absolutely. All analysis happens on-device using Apple's privacy-focused frameworks. No usage data leaves your phone.

Can I customize when nudges appear?

Yes! Set quiet hours, adjust sensitivity, choose different nudge styles, and select which apps to monitor.

Will this drain my battery?

No. Nudge uses efficient background monitoring that's designed to have minimal impact on battery life.

Which iOS version do I need?

Nudge requires iOS 16.0 or later to access the advanced Screen Time API features needed for real-time monitoring.

How is this different from Screen Time limits?

Instead of blocking apps after time limits, Nudge provides gentle awareness cues during usage to promote mindful decision-making.
